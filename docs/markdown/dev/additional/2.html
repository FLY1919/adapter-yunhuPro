3. 获取历史消息 ​

接口描述

获取指定会话中的历史消息。

请求 URLhttps://chat-go.jwzhd.com/open-apis/v1/bot/messages
请求方法GET

Query 参数

参数类型必须描述
tokenstring是机器人的 token
chat-idstring是会话 ID (群组或用户)
chat-typestring是会话类型 (group 或 bot)
message-idstring是起始消息的 ID
beforenumber是获取该消息之前的消息数量
afternumber是获取该消息之后的消息数量

注意： before 和 after 不能同时为 0。

响应示例 (Response)

点击展开/折叠
json
{
    "code": 1,
    "data": {
        "list": [
            {
                "msgId": "30e9dad15b5244f9a1465b4f6e7949d8",
                "senderId": "7756242",
                "contentType": "text",
                "content": { "text": "饿啊" },
                "sendTime": 1761136474991
            },
            {
                "msgId": "f8190054dde84502917c2e7d1f9deec0",
                "senderId": "7756242",
                "contentType": "text",
                "content": { "text": "找到了src" },
